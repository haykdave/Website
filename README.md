# Hayk Davtyan Portfolio

Personal portfolio website presenting my background, technical skills, selected projects, and contact information.

## Tech Stack

- HTML5
- CSS3
- JavaScript (vanilla)

## Project Structure

- `Portfolio/index.html` - Main portfolio page
- `Portfolio/style.css` - Styling and responsive design
- `Portfolio/script.js` - Interactions, animations, theme toggle, and form behavior

## Run Locally

1. Clone the repository.
2. Open `Website/Portfolio/index.html` in your browser.

Optional: use a local server if you prefer, for example:

```bash
cd Website/Portfolio
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Features

- Responsive layout for desktop and mobile
- Sticky/fixed header with adaptive behavior
- Dark/light theme toggle
- Project showcase with external links
- Direct email link via `mailto:`

## macOS Mailto Note

On macOS, clicking email links may not open a compose window if no default email app is configured.

If `mailto:` links do not open:

1. Open the **Mail** app.
2. Go to **Mail > Settings**.
3. In **General**, set **Default email reader** to **Mail** (or your preferred mail client).
4. Try the email link again.
